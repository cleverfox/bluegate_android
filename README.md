# BlueGate

An Android application for control *bluegate* bluetooth garage door or gate controller

## Features
- Authenticate on bluegate controller
- Bluegate controller management: keys, settings, logs

## Permissions
 I requires bluetooth and location permissions, it use it to connect over BLE. App does not require internet access.
 To exchage public keys you can you your favorite messenger, this app will interact over clipboard
 
## Setup
1. Clone the repository.
2. Open in Android Studio.
3. Build and run on your device.
